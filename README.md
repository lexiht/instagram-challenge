Instagram Challenge
===================

A weekend challenge to build an instagram clone.

Instructions
-------

* `git clone https://github.com/lexiht/instagram-challenge.git`
* `bundle install`
* `rake db:setup`
* `rails s`
* go to `localhost:3000`
